Как работать с Git

Инициализация репозитория и первый коммит:
git init
git add expense_tracker.py
git commit -m "Initial commit: Expense Tracker app"
Игнорирование tmp-файлов (если есть) и сохранение настроек:
Создайте файл .gitignore и добавьте: *.pyc, pycache/
Частое добавление изменений:
git add.
git commit -m "Добавлен функционал фильтрации и JSON-импорт/экспорт"
Расширения и улучшения (идеи)

Поддержка сохранения состояния в локальный JSON-файл по умолчанию.
Блокировка редактирования выбранной записи.
Гибкие фильтры по нескольким категориям и диапазону дат.
Валидация сумм с поддержкой локализации (десятичный разделитель).
Подключение к SQLite для долговременного хранения.
Если нужно, могу адаптировать код под PyQt5/6 или другую GUI-библиотеку, добавить экспорт в CSV, графики расходов и тесты.
